RBES is a prototype for a new kind of English school.
Instead of sitting in classrooms, students create imaginary startups which become real.
Instead of taking tests, they design roller coasters, create animated films, publish newspapers, craft REIT proposals, write sitcoms, research the optimal Pad Thai, architect tunnelparks, code games, and paint collaboratively.

RBES is not (just) a performance-art piece; it is meant to be a real enterprise.
This playbook is both for future student-owners and investors.

On it, you’ll learn:
(1) Who we are,
(2) The popup workshops running now,
(3) The year-long test of concept,
(4) The theory behind the model,
(5) The revenue model,
(6) The budget,
(7) The metrics and outcomes,
(8) The intellectual context, and
(9) The shared ownership program.
